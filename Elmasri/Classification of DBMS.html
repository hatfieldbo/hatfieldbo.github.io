Classification of DBMS

  • by data model

    • Relational DBMS - represents a database as a collection of tables

    • Object DBMS - represents a database in terms of objects, their properties, and their operations (methods)

    • Object-relational DBMS - relational DBMS with extended object database concepts

    • Hierarchical DBMS - represents a database as hierarchical tree structures

    • Network DBMS - represents data as record types

  • by number of users
    • single-user systems
    • multi-user systems
  • by number of sites
    • centralized DBMS
    • distributed DBMS (DDBMS) - use client-server architecture
      • Homogeneous DDBMS
      • Heterogeneous DDBMS
      • Federated DBMS (or multidatabase system)
  • by cost
    • single-user low-end systems
    • multi-user high-end systems
  • by the purpose
    • general-purpose 
    • special-purpose (e.g., OLTP - On-Line Transaction Processing system)
